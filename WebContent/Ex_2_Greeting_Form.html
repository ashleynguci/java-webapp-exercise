HTML Form + Servlet

When the button is pressed, the browser sends an HTTP POST request with the form data to the Web server. The Web server sends an HTML document to the browser as the response.

Form parameters are passed to the servlet as name/value pairs in an HTTP POST request. The servlet container invokes the servlet's doPost method when the browser sends a HTTP POST request to the servlet. You can use the HttpRequest object's getParameter method to obtain the parameter values. The method returns either the parameter value as a String, or null if the parameter is not part of the request.
