HTML Form / Sending an HTTP request

1. When you click on the link below, then the browser sends an HTTP GET request for the Web resource. The Web server sends an HTML document to the browser as the response.
    http://localhost:8080/First_WebApp_Examples/firstServlet

2. When you click on the link below, then the browser sends an HTTP GET request for the Web resource. This time, the request includes two parameters.
    http://localhost:8080/First_WebApp_Examples/secondServlet?firstname=Joe&title=Mr
